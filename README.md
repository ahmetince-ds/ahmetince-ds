# 📊 Customer Segmentation & CLTV Analysis (RFM + BG/NBD + Gamma-Gamma)

## 📌 Project Overview
This project focuses on customer segmentation and Customer Lifetime Value (CLTV) prediction using the Online Retail dataset.  
RFM analysis and probabilistic models (BG/NBD & Gamma-Gamma) are applied to understand customer behavior and predict future value.

---

## 🧠 Objectives
- Segment customers using RFM analysis
- Identify high-value customers
- Predict future purchase behavior
- Estimate Customer Lifetime Value (CLTV)

---

## 📂 Dataset
Online Retail II dataset containing:
- Invoice data
- Product details
- Customer transactions
- Quantity & Price information

---

## ⚙️ Methods Used

### 1. Data Preprocessing
- Missing value handling
- Filtering cancelled invoices
- Feature engineering (TotalPrice)

### 2. RFM Analysis
- Recency: last purchase date
- Frequency: number of purchases
- Monetary: total spending
- Customer segmentation (Champions, Loyal, At Risk, etc.)

### 3. CLTV Prediction
- BG/NBD Model → purchase frequency prediction
- Gamma-Gamma Model → profit estimation
- Final CLTV calculation

---

## 📊 Key Insights
- High-value customers identified
- Customer segments clearly separated
- Future purchase behavior estimated
- Business-oriented customer targeting possible
